There are multiple different configurations for the visualization:
* Wheels with or without directional flags
* Base with or without bumpers
-----------------------------------------------------------------
A complete configuration needs a file for the base and 4 wheel files such that:

 -	Robot_Swerve.glb
 -	Robot_Swerve_0.glb
 -	Robot_Swerve_1.glb
 -	Robot_Swerve_2.glb
 -	Robot_Swerve_3.glb

...are all present.
